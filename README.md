Personal Light Streaming  
====================
####PLS  
  
Development of a personal web music streaming plateform.  
It has to be light enough to run on a Raspberry Pi with a web server (preferably Apache, and if too heavy, nginx).  
  
 * * *  
  
Used technologies
-----------------------
 * HTML 5
 * CSS 3
 * PHP 6
 * JavaScript
 * Probably Ajax
  
  
Used add-ons
-----------------
 * [Bootstrap][bs]
 * [jQuery][jq]
 * [SoundManager 2][sm2]
